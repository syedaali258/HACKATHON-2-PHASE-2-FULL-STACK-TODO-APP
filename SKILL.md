# Frontend Skill – Pages, Components & Layouts

## Instructions

1. **Page structure**
   - Semantic HTML structure
   - Header, main, footer sections
   - Responsive container system

2. **Components**
   - Reusable UI components
   - Props-based customization
   - Consistent spacing and typography

3. **Layouts**
   - Flexbox and Grid usage
   - Responsive breakpoints
   - Sidebar, navbar, and content layouts

4. **Styling**
   - Modern CSS (Flexbox, Grid)
   - Utility-first or component-based styles
   - Hover, focus, and active states

## Best Practices
- Mobile-first design
- Reusable and modular components
- Consistent color and spacing system
- Avoid inline styles
- Use semantic HTML elements

## Example Structure
```html
<div class="page">
  <header class="navbar">
    <h1 class="logo">Brand</h1>
  </header>

  <main class="layout">
    <aside class="sidebar">Menu</aside>
    <section class="content">
      <div class="card component">
        <h2>Component Title</h2>
        <p>Reusable component content</p>
      </div>
    </section>
  </main>

  <footer class="footer">© 2026</footer>
</div>
